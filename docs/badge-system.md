# Badge System

## Badge Types
- Completion badges
- Mastery badges
- Milestone badges

## Issuance Rules
- Learner meets XP threshold
- Required assessments passed
- Signature or on-chain verification completed

## Data Model (High-Level)
- `badge_id`
- `learner_id`
- `track`
- `issued_at`
- `verification_ref`
