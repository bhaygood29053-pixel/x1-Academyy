# Product Brief — X1 Academy

## One‑liner
Learn X1 + Xen by playing: interactive stories, quizzes, puzzles, and badge rewards.

## Primary users
- Newcomers to X1 who want a guided, fun path
- Existing community members who want structured knowledge + sharable achievements

## Core value
- Turns X1 concepts into bite‑size challenges
- Builds confidence and correct mental models
- Rewards progress with XP + badges

## What makes it "interactive"
- Story checkpoints: choose‑your‑path + comprehension questions
- Quizzes: randomized question order + instant feedback
- Puzzles: ordering steps, matching terms, “spot the bug” mini challenges

## Reward model
- XP for completing lessons + quizzes
- Badges for milestones (module completion, streaks, perfect scores)

## MVP scope (v0.1)
- Wallet connect gate (sign message to create a profile)
- Curriculum list + lesson viewer
- Quiz runner + scoring
- Progress tracking + badge shelf

## Non‑goals (for MVP)
- Full on‑chain badge minting
- Real token rewards
- Multiplayer / PVP
- Complex economies

## Phase 2+ options
- On‑chain attestations (cheap proofs of completion)
- Badge NFTs / compressed NFTs
- Community “quests” and seasonal events
