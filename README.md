# x1-Academyy

Monorepo scaffold for x1-Academy.

## Repository Layout

- `docs/` – product, curriculum, badge, security, cost, and roadmap docs
- `apps/web/` – Next.js web application
- `packages/content/` – lessons, question banks, and badge content
- `packages/ui/` – shared UI package
- `packages/sdk/` – TypeScript helpers (score, XP, signatures, tx builders)
- `services/api/` – Fastify/Express backend service
- `programs/badges/` – Anchor badge programs (Phase 2+)
- `infra/ci/` and `infra/docker/` – operational infrastructure
- `scripts/` – automation scripts
